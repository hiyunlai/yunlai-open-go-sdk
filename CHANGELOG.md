# 更新日志

## [1.0.3] - 2025-11-28

### 新增
- **API**: 添加获取分享投注方案接口
  - 新增 `GetShareBetOrder` 方法，用于获取分享的投注方案信息
  - 新增 `ShareBetOrder` 数据结构，包含彩种类型、子玩法、期次、倍数、过关方式、投注码等字段
  - 新增 Order 业务模块

### 文档
- **README**: 更新业务模块说明，新增 Order 模块文档

## [1.0.2] - 2025-11-05

### 重构
- **API**: 为状态和状态码使用特定类型
  - 将 API 结构体中的 `Status` 和 `StatusCode` 字段从通用的 `string` 类型更改为自定义的 `Status` 和 `StatusCode` 类型
  - 在 `constants.go` 中定义了 `Status` 类型及其相关常量，提高代码的可读性和类型安全性
